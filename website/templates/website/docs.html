API Documentation

The ANIMATOR AI API allows you to create and retrieve animated videos by uploading images and driving videos.

    How to get started with the ANIMATOR AI API:

POST:

Create Media

API: POST Media Create View /api/media/

Endpoint: /api/media/

Description: Uploads an image and a driving video, processes them to create an animated video, and returns the URL of the animated video.

Request:

Form Data:
    {
        "driving_video": File (required),
        "input_image": Image (required)
    }

Response:

    201 Created
    {
        "id": 1,
        "driving_video": "http://127.0.0.1:8000/media/driving_videos/your_video.mp4",
        "input_image": "http://127.0.0.1:8000/media/input_images/your_image.jpg",
        "output_video": "http://127.0.0.1:8000/media/output_animations/output_1.mp4",
        "created_at": "2024-08-03T12:34:56Z"
    }

GET:

Get Media List

API: GET Media List View /api/media/list/

Endpoint: /api/media/list

Description: Retrieves a list of all media entries.

Request:

Form Data:
    {
        "driving_video": File (required),
        "input_image": Image (required)
    }

Response:

    200 OK

Create Media:

Endpoint: /api/media/

Description: Upload an image and a driving video to initiate the creation of an animated video. The API processes the input files using a trained AI model implemented with PyTorch. The model uses the driving video to drive the animation of the input image, generating a new animated video. The response includes the URLs for the original files (driving video and input image) and the URL for the resulting animated video. This endpoint leverages advanced AI techniques to produce high-quality animations based on user-provided media

API: GET Media List View /api/media/

List Media:

Endpoint: /api/media/list

Description: Retrieve a list of all media entries stored in the system. Each entry includes the URLs for the driving video, input image, and the generated animated video, along with the timestamp of when the media was created. The list provides an overview of all processed media, allowing you to manage and review previously uploaded files and their associated animations. This endpoint showcases the results of the AI model's processing, implemented using PyTorch, across all media entries
